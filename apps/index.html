Project LITE Apps



“Project LITE – Light Inquiry Through Experiments” was an NSF supported STEM educational project at Boston University (Kenneth Brecher, Principal Investigator). The web site for the entire project is:
http://lite.bu.edu/

Part of the project explored many aspects of dichoptic or binocular vision. Below is a link to the Project LITE visual perception apps that were developed for viewing on smartphones.
http://lite5.bu.edu/v4.1/
